# Backlight

Sometimes you need a light source for reading a paper, having a video call in a dark room, or just to reduce eye strain. So this is a simple app that turns your screen into a backlight with adjustable brightness and color temperature.

## Features

- Brightness: to change brightness, use the up/down arrows or swipe up/down on mobile;
- Temperature: to change temperature, use the left/right arrows or swipe left/right on mobile.
- Custom colors: you can also pick any color you want, just click on the color box to open a color picker and choose any color you like.
